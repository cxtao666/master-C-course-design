# Student Grade Management System

A C course-design project for managing student records and course grades.

## Features

- Enter and persist student information
- Calculate totals and averages
- Sort records in ascending or descending order
- Search by student number or name
- Add, edit, and remove records
- Generate basic grade statistics

## Build

The source targets Windows and uses `windows.h`. Compile it with a compatible C toolchain:

```bash
gcc 学生成绩管理.c -o grade-manager
```
